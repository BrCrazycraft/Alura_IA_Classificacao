Api REST

Api de constulta de banco de dados de usarios feito em python, a lista possui usarios com suas posições e votos

Exemplo de uso

Exemplos de como fazer solicitação da API

Python

Instale a biblioteca [requests] usando o CMD

Terminal

pip install requests

Após fazer a instalação você precisara importação você pode fazer dois meios de solicitação

1. URL

solicitação via URL são buscas gerais no Banco de dados

Python

import requests

#String com o caminho do site mais a sua solicitação
url = "caminho/solicatacao"

#Variavel que guarda a referencia, da sua solicitação
response = requests.get(url)

#Verifica se a requisição foi um sucesso
if response.status_code == 200:

    #JSON com a resposta do servidor
    data = response.get_json()

2. Post

Solicitação post você faz requisições para a API, envia um JSON e recebe um JSON como resposta

Python

import requests

#String com o caminho do site mais a sua solicitação
url = "caminho/solicatacao"

#Um dicionario com a mensagem que você vai enviar ao servidor
request = {
    "research": {
        "repo": "Poneglyph",
    },
}

#Variavel que guarda a referencia, da sua solicitação
response = requests.get(url)

#Verifica se a requisição foi um sucesso
if response.status_code == 200:

    #JSON com a resposta do servidor
    data = response.get_json()

Se você quiser da print na solicitação e so escrever print(data)
